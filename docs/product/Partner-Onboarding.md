# LetLogic Partner Onboarding Playbook

**Linear:** [AGE-71](https://linear.app/agentscale/issue/AGE-71)  
**Repo:** `/Users/scottmaryan/Desktop/Letlogic`

## Week 0 — Kickoff

- [ ] Sign or email-confirm pilot agreement
- [ ] Create account (magic link)
- [ ] Grant comp credits via admin adjustment (`credit_ledger`, reason `adjustment`)
- [ ] Walkthrough: `/sample` → `/screen` → add property → compare view
- [ ] Share support email: support@letlogic.app

**Kickoff questions:**
1. How do applications arrive today (email, form, agent)?
2. What would make you pay £4.99+ per screening?
3. What would make you *not* use this?

## Weeks 1–3 — Weekly check-in (15 min)

| Week | Focus questions |
|------|-----------------|
| 1 | First screening done? What surprised you? |
| 2 | Used comparison? Re-screened anyone? |
| 3 | Would you buy credits when comp runs out? Why/why not? |

**Track in Linear:** screenings count, properties added, comparison usage.

## Week 4 — Exit interview

- [ ] What did LetLogic catch that you might have missed?
- [ ] What was wrong or missing?
- [ ] Would you recommend to another landlord? (1–10 NPS)
- [ ] Permission for testimonial / case study?
- [ ] Offer: first paid pack at discount if converting

## Success criteria

- 5+ real screenings in 4 weeks
- At least one quotable testimonial
- Documented feature requests for M4 roadmap
